Data Visualization and Interactive Dashboards with Python

Tutorial Book - 2022

This book walks the reader through from installing python to deploying their own dashboard onto the web. You will be able to get familiar with many essential python libraries and other platforms such as Git/GitHub and Heroku while learning about important topics such as data manipulation, data visualization, and application deploy.

The book also comes with supporting code and resources that will allow the readers to get a deeper understanding of each topic discussed throughout the book.
